# mlb_feed
MLB Wrapper API for gd2.mlb.com


# Routes

## Games
- GET /games/year/:year
- GET /games/team/year/:year

- GET /games/month/:year/:month
- GET /games/team/month/:team/:year/:month

- GET /games/day/:year/:month/:day
- GET /games/team/day/:team/:year/:month/:day

## Teams
- GET /team
- GET /team/:id
